# 🛒 쇼핑몰 웹 애플리케이션 (Spring Boot)

스프링부트를 기반으로 개발한 **쇼핑몰 웹 프로젝트**입니다.  
회원 가입, 상품 조회, 장바구니, 주문 기능을 포함하며, MySQL과 MyBatis 기반으로 동작합니다.

## 기술

| 기술 | 설명 |
|------|------|
|  Back-end | Spring Boot, Spring MVC, MyBatis |
|  DB | MySQL 8.x |
|  Front-end | JSP, JSTL, Bootstrap |


---

##  주요 기능

###  회원
- 회원 가입 / 로그인
- 고객 정보 수정
- 관리자 계정 분리

### 상품
- 카테고리별 상품 조회
- 상품 상세 보기
- 할인율 적용
- 이미지 업로드/수정

### 장바구니
- 상품 추가 / 삭제 / 수량 변경
- 장바구니 목록 보기

###  주문
- 주문 생성
- 주문 상세 정보 확인
- 가상 결제 (고객 잔액 차감 방식)

---

## 💾 DB 테이블

- cate (카테고리)
- cust (고객)
- product (상품)
- cart (장바구니)
- order (주문)
- orderdetail (주문 상세)

>  제약조건을 통해 데이터 정합성 유지
